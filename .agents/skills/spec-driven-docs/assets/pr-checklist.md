## Documentation check

- [ ] No persistent plans, task lists, scratch docs, or archive/active/done folders added.
- [ ] Durable behavior change → updated relevant file under `docs/specs/`.
- [ ] Major architectural decision → added or updated ADR under `docs/decisions/`.
- [ ] System map changed → updated `docs/system.md`.
- [ ] If no docs touched, the change does not alter durable behavior, contracts, architecture, or system structure.
